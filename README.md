# Financial RAG Thesis

## Objective
Build and evaluate a Retrieval-Augmented Generation (RAG) system for financial question answering.

## Setup
```bash
python -m venv venv
source venv/bin/activate   # or venv\Scripts\activate (Windows)
pip install -r requirements.txt
```

## Structure
- data/: datasets
- src/: core code
- notebooks/: experiments
- experiments/: results tracking
